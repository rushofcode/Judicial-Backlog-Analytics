# Judicial-Backlog-Analytics
## Project Overview
This project analyzes judicial infrastructure and case clearance efficiency across Indian states using state-level performance metrics.

## Business Questions
- Which states face the highest courtroom infrastructure shortages?
- How do High Court and Lower Court clearance rates compare?
- Is judicial manpower proportional to population burden?
- Does higher judicial spending improve efficiency?

## Dataset Source
State-level judicial capacity and clearance data (2022)
